# Portfolio — Mostafa Cherif

Sitio estático de presentación de **servicios de edición de vídeo** (YouTube horizontal, formato vertical y miniaturas), con enfoque en retención y narrativa.

**Stack:** HTML5, CSS y JavaScript vanilla. Tema claro/oscuro, diseño responsive y secciones accesibles (landmarks, etiquetas ARIA en el carrusel de miniaturas).

**Repo:** [github.com/mcherif004/Portfolio-Video](https://github.com/mcherif004/Portfolio-Video)

**GitHub Pages:** `https://mcherif004.github.io/Portfolio-Video/`

**Si no carga:** en el repo → **Settings → Pages → Build and deployment → Source** elige **GitHub Actions** (no “Deploy from a branch” a la vez). En **Actions → General** deja **Workflow permissions: Read and write**. Si el job pide aprobación del entorno `github-pages`, acéptala una vez. Alternativa sin este workflow: Source **Deploy from a branch**, rama **main**, carpeta **/** (raíz).

Para verlo en local: abre `index.html` en el navegador o sirve la carpeta con cualquier servidor estático.
